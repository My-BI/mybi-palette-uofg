# University of Glasgow — MyBI colour palette

A colour palette for [MyBI](https://github.com/My-BI), in the open GIMP `.gpl` format.
It has **Primary / Secondary / Additional (functional)** brand colour groups plus
**light / dark / OLED** app-theme sets, so selecting it themes both your charts and
the app chrome.

## Use in MyBI

**Colour palettes** (left rail) → **Marketplace** → **University of Glasgow** → **Install**.
Turn on **Auto-update** to keep it in sync with this repo's `colour-palette` release.
Or download `mybi-uofg.gpl` from the [latest release](../../releases/latest) and import it
(**Colour palettes → ⤓**).
